##  Objetivo del programa
Desarrollar un generador de contraseñas seguras que permita crear claves fuertes
de forma automática, aleatorias y personalizables, aplicando los conceptos
aprendidos en las cuatro unidades de la asignatura de Lógica de Programación.

## Introducción al proyecto
Durante las ocho semanas del curso se estudiaron los fundamentos de la resolución
de problemas mediante programación, el manejo de datos, el uso de algoritmos,
estructuras de control, de datos y funciones. 

Consiste en una aplicación que se encargue de generar contraseñas seguras para los usuarios
para facilitar la seguridad y protección de datos personales, incluyendo funciones como elegir la longitud de la contraseña, que tipos de caracteres puede utilizar como (mayúsculas, minúsculas, números y símbolos) y verificar la seguridad de las contraseñas.

Este proyecto integra dichos conocimientos en una aplicación práctica que resuelve
Un problema real: la generación de contraseñas seguras para la protección de la
información personal de los usuarios como barrera para evitar a personas con malas intenciones.

##  Funcionalidades principales
- Generación automática de contraseñas seguras
- Selección de longitud de la contraseña
- Inclusión de diferentes tipos de caracteres:
  - Letras mayúsculas
  - Letras minúsculas
  - Números
  - Caracteres especiales
- Validación de datos ingresados por el usuario
