## Objetivo del proyecto
Desarrollar un software que nos permita generar contraseñas de manera segura, aleatorias y personalizables,
aplicando de esta manera los conocimientos que fueron adquiridos durante las 4 unidades de la asignatura de Lógica de programación

## Descripción del proyecto
Este proyecto consiste en una aplicación que se encargue de generar contraseñas seguras para los usuarios
para facilitar la seguridad y protección de datos personales, incluyendo funciones como elegir la
longitud de la contraseña, que tipos de caracteres puede utilizar como (mayúsculas, minúsculas, números y 
símbolos).

## Funcionalidades principales
- Generación de contraseñas seguras
- Fortaleza de contraseña (fácil, media, difícil)
- Guardar contraseñas
